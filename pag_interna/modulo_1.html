Módulo 01

Front-end, Back-end e Full stack

Front-End: Responsável pelo client-side (lado do cliente), pela parte visual e interação do usuário, através de HTML, CSS e JavaScript.

Back-End: Responsável pelo server-side (lado do servidor), é a linguagem que vai rodar "por trás dos panos".

Full Stack: É o desenvolvedor quer tem o conhecimento tanto do client-side e do server-side.

Linha Horizontal ex001

Para se criar uma linha horizontal, como a que está separando um conteúdo dos outros, é usada a tag <hr> (não possui fechamento).

Parágrafos e Quebra de Linhas ex002

Parágrafos são formados por textos escritos dentro das tags <p>"Parágrafo aqui"</p>.

Já a quebra de linha é através da tag <br> (não possui fechamento).

Adicionando Imagens ex003

Para adicionar uma imagem ao seu HTML, usa-se a tag <img src="diretório"> alt="especificar a imagem (é ideal para o navegador identificar sobre o que está se falando)">.

Favicon ex004

Para adicionar um favicon (favorite icon), acima do <title></title> é usado o código: <link rel="shortcut icon" href="diretório.ico" type="image/x-icon">.

Hierarquia de Títulos

Os títulos vão de h1 ao h6, basta escrever seus títulos dentro do <h#>.

Formatação de Textos ex008

Formatações semânticas quer dizer que são formatações que visualmente não mudarão nada da não semânticas, mas darão sentido a sua forma, como por exemplo o negrito "strong" (semântica), "b" (não semântica).

Formatações não semânticas

<b> - Negrito
<i> - Itálico
<u> - Sublinhado

Fromatações semânticas

<strong> - Negrito
<em> - Itálico (em = ênfase)
<mark> - Marcador de texto
<del> - Texto deletado
<small> - Texto pequeno
<sup> - Texto subscrito
<sub> - Texto sobescrito
<ins> - Texto inserido

Outras Formatações ex008b

Código

<code> - Inserior o texto como um código.

<pre> - Criar uma formatação pré-formatado. Geralmente colocado antes dos <code

Citações

<q> - Dá sentido a uma citação

Citações completas

Segundo "Fulano" <blockquote cite="link da citação aqui">"Texto aqui"</blockquote>

Não gera nenhum efeito visual, mas faz o navegador reconhecer.

Abreviações

<abbr title="TEXTO_COMPLETO_AQUI">TXTCA</abbr>

Ao passar o mouse por cima, vê-se o título da citação.

Trabalhando Listas ex009

<ol> - Listas ordenadas
<ul> - Listas não ordenadas <!--disc circle square-->
<li> - listas

<dt>"Título da Lista de definição"</dt>
<dd>"Escritura dentro do conteúdo da lista"<dd>

Links para Downloads Externos e Internos ex010

Para encaminhar um usuário a um link, seja externo ou interno, é usada a tag <a href="#" target="_#" rel="#">"NOME"</a>.

Estilos Inline ex013

<h1 style= "color: blueviolet"> Capítulo 1</h1>

Desta forma acima, são aplicados o css dentro de uma tag

Estilos locais ou internos ex014

Abaixo do title, dentro do head, devemos usar a tag <style>, e dentro colocar todos nossos códigos em css para a página.

Estilos externos ex015

Criamos uma novo código em css, e dentro do head adicionamos um <link rel="stylesheet" href="#.css">, e no código de css fazemos as alterações.
