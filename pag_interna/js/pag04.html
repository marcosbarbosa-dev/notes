DOCUMENT OBJECT MODEL (DOM)

DOCUMENT OBJECT MODEL (DOM)

DOCUMENT OBJECT MODEL (DOM)

DOM é um modelo de objetos para documentos. É um conjunto de objetos dentro do seu navegador que dá acesso aos componentes internos do seu website.

ÁRVORE DOM

ÁRVORE DOM

Tudo dentro do JavaScript está dentro de 'window'.
Dentro do window temos: location, document, history.
location: diz qual é a localização do seu site, como, qual é página atual, qual é a url, qual foi a página anterior.
document: documento atual.
history: vai guardando de onde você veio, pra onde você vai.

Dentro do document temos o objeto: html (que é toda parte de html do site);

Dentro de html temos dois filhos (child) head e body: cabeçalho e corpo.
Vale notar que o head e body são filhos de html, já o html é o parent, um pai/mãe, para o head e body.

Dentro de head temos todas tags que podemos colocar dentro do cabeçalho, como meta, title, etc...
Dentro do body podemos ter diversas tags, como, h1, p, div, etc.
Dentro de um parágrafo, por exemplo '<p>Nome <strong>completo</strong> aqui</p>', teremos um child, que é o strong.

EVENTOS DOM

EVENTOS DOM

Evento é tudo aquilo que pode acontecer com um elemento, sendo o mais comum, eventos de mouse.

Quando entrarmos com o mouse dentro do elemento, que aqui chamarei de div, será disparado o evento 'mouseenter', e quando continuarmos movendo o mouse já dentro da div, será disparado o evento 'mousemove'. Quando clicarmos e segurarmos, teremos o disparo do evento 'mousedown', quando soltarmos o botão do mouse, teremos o evento 'mouseup'. Quando clicamos rapiamente e soltamos, temos o evento 'click', e quando retirarmos o mouse da div 'mouseout'.

mouseenter // ao entrar com mouse em cima do elemento
mousemove // ao mover o mouse dentro do elemento
mousedown // ao clicar e segurar com mouse dentro do elemento
mouseup // ao soltar o clique do mouse dentro do elemento
click // ao clicar rapidamente com mouse dentro do elemento
mouseout // ao retirar o mouse de cima do elemento

FUNÇÃO em javascript é conjunto de códigos que vão ser executadas somente quando o evento ocorrer.

Digamos que teremos 10 linhas de códigos, essas 10 linhas serão chamadas de bloco. Ele só ocorrerá por exemplo, quando eu clicar dentro da div. Para que isso aconteça, devemos colocar as linhas dentro de um bloco
