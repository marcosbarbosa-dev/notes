ADICIONANDO O NOME A UMA VARIÁVEL

Antes do código windows.prompt(''), podemos usar o 'var nome =', para que o que for digitado entre parenteses, vire uma variável chamada 'nome'.
Exemplo: var nome = window.prompt('Qual é o seu nome?')

CONCATENAÇÃO

CONCATENAÇÃO

O sinal de + é uma concatenação, usado para juntar uma coisa a outra.

Exemplo:
var nome = window.prompt('Qual é o seu nome?') - respondo: Marcos
windows.alert('Olá' + nome) Isso aparecerá para o cliente como 'Olá Marcos', pois Marcos se tornou a variável de 'nome', e fora da aspas, adicionei através da concatenação, a variável ao 'Olá'.

CONVERSÕES DE STRING PARA NUMBER

CONVERSÕES DE STRING PARA NUMBER

Number.parseInt(n) - ou apenas .parseInt(n), converterá a string para um número inteiro.
Number.parseFloat(n) - ou apenas .parseFloat(n), converterá a string para um número real, ou seja, um número com vírgula.
Lembrando que devemos respeitar as letras maiúsculas e minúsculas.

EXEMPLO:
var n1 = Number.parseInt(window.prompt('Digite um número:'))
var n2 = Number.parseInt(window.prompt('Digite outro número:'))

Assim, a varíavel n1 e n2, irá receber o que vier do prompt, aonde está escrito para digitar o número, convertido para um número inteiro.

Caso não seja feita a conversão, o javascript verá os prompts como strings, e ao invés de em caso de somar as variáveis (caso tenha adicionado a função de adição e criado um alert para a resposta), dará a resposta como a junção de dois números. Exemplo: 5+4 = 54.

CONVERSÕES DE STRING PARA NUMBER FACILITADO (ECMAScript)

CONVERSÕES DE STRING PARA NUMBER FACILITADO

Ao invés de usarmos o parseInt ou parseFloat, podemos usar apenas o Number(n), e o javascript descobrirá se é real ou inteiro.

EXEMPLO:
var n1 = Number(window.prompt('Digite um número:')) var n2 = Number(window.prompt('Digite outro número:')) Mas ainda é válido usar o parseInt e parseFloat em algumas situações, caso queiramos forçar obrigatóriamente o resultado ser real ou inteiro.

TEMPLATE STRINGS

TEMPLATE STRINGS

Digamos que criamos 3 variáveis:
var nome = 'Marcos'
var idade = 22
var nota = 5.5

E queremos escrever a mensagem 'O aluno Marcos de 22 anos tirou nota 5.5'. Do jeito tradicional usaríamos o seguinte modo:
('O aluno ' + nome + ' de ' + idade + ' anos tirou nota ' + nota)

No Template String, para facilitar, usaremos deste modo:
(`O aluno ${nome} de ${idade} anos tirou ${nota}`)
Usando a crase para abrir e fechar. Ela é usada no template string.
O ${} é chamado de placeholder.

TESTANDO NOVOS COMANDOS

TESTANDO NOVOS COMANDOS

var nome = window.prompt('Qual é o seu nome?')
document.write(`Seu nome tem ${nome.length} tantas letras`)

O que aconteceu é que, ao definir a variável nome, pedimos para que o JS escrevesse na tela a quantidade de caracteres que meu nome tem, através do .length

Podemos também usar tags HTML dentro do código.

EXEMPLO:
var nome = window.prompt('Qual é o seu nome?')
document.write(`<h2>Seu nome tem ${nome.length} tantas letras</h2>`)

OU

var nome = window.prompt('Qual é o seu nome?')
document.write(`Olá <strong>${nome}</strong>! Seu nome tem ${nome.length} tantas letras.`)
document.write(`Seu nome em maiúsculo é ${nome.toUpperCase()}`)

FORMATANDO NÚMEROS

FORMATANDO NÚMEROS

.toFixed(n) // Em caso de números variáveis, podemos adicionar a quanitdade de casas decimáis.

EXEMPLO:
var n1 = 1454.5 // n1 = 1454.5
var n1 = 1454.5 // n1.toFixed(3) = 1454.500

.toFixed(n).replace('.', ',') // Escolhi a quantidade de casas decimáis e fiz o replace (alterei) o ponto por vírgula.

var n1 = 1454.5 // n1.toFixed(3).replace('.', ',') = 1454,500
