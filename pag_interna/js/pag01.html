IDENTIFICADORES

IDENTIFICADORES

  • Podem começar com letra, $ ou _.
  • Não podem começar com números.
  • É possível usar letras ou números.
  • É possível usar acentos e símbolos.
  • Não podem conter espaços. São substituído pelo sinal de espaço _.
  • Não podem ser palavras reservadas. Exemplo: function, alert, var.

Dicas para nome de identificadores

Dicas para nome de identificadores

Variáveis com letras maiúsculas e minúsculas, fazem diferença.
Exemplo de duas variáveis que serão diferentes com a mesma letra:
var a = 'Marcos' var A = 'Antônio'

Variáveis do tipo NUMBER

Variáveis do tipo NUMBER

Para o JavaScript, o tipo Inteiro e o Real, são todos do mesmo tipo (number).

Variáveis de tipo INTEIRO:
5 18 -12 (não possuem partes fracionárias).
Variáveis do tipo REAL:
0.5 -15.9 3.14 8.0 (são números com ponto flutuante, chamados de float).

Variáveis do tipo STRING (cadeias de caracteres)

Variáveis do tipo STRING (cadeias de caracteres)

Uma string também pode ser um conjunto de números, como CPF, Idt, Tel.

Variáveis do tipo BOOLEAN: true false
