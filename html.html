Primeiros Passos Teorícos

Qual a diferença entre Front-end, Back-end e Fullstack?

Front-End: Responsável pelo client-side (lado do cliente), pela parte visual e interação do usuário, através de HTML, CSS e JavaScript.
Back-End: Responsável pelo server-side (lado do servidor), é a linguagem que vai rodar "por trás dos panos".
Full Stack: É o desenvolvedor quer tem o conhecimento tanto do client-side e do server-side.

TAGS

O que são TAGS?

HTML é uma parte de uma página da internet. Um elemento é qualquer objeto identificável dentro de um documento, por exemplo, um caractere, palavra, imagem, parágrafo ou célula de planilha.

hr : Linha horizontal.
br : Quebra de linha.
strong : Negrito.
em : Itálico (em = ênfase).
mark : Marcador de texto.
del : Texto deletado.
small : Texto pequeno.
sup : Texto subscrito.
sub : Texto sobescrito.
ins : Texto inserido.
code : Cria um estilo visual de código.
a href"#" : Encaminha um usuário a um link.

LISTAS:
ol : Listas ordenadas.
ul : Listas não ordenadas - disc circle square.
li : listas.
dt : Título da Lista de definição.
dd : Escritura dentro do conteúdo da lista.

TÍTULOS: os títulos vão de h1 ao h6, basta escrever seus títulos dentro do <h#>.

Estilos - Inline, Local ou Interno e Externo

O que são estilos?

CSS é um mecanismo para adicionar estilo a um documento web.

Inline: são estilos (css) aplicado dentro da tag.
Exemplo: <h1 style= "color: blueviolet"> Capítulo 1</h1>

Local ou Interno: abaixo do title, dentro do head, devemos usar a tag <style>, e dentro colocar todos nossos códigos em css para a página.

Externo: criamos uma novo código em css, e dentro do head adicionamos um <link rel="stylesheet" href="#.css">, e no código de css fazemos as alterações.
