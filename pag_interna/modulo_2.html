Fontes em CSS ex017

Adicionamos dentro do nosso CSS o body {font-family: 'fonte';}. Importante ressaltar que quando a fonte possuir duas ou mais palavras, devem estar dentro de aspas simples.

Exemplo:
body { font-family: Arial; }.
body { font-family: 'Times New Roman'; }

Geralmente usamos o seguinte tipo de declaração, chamada de safe combination (combinação segura):
body { font-family: Arial, Helvetica, sans-serif; }, pois, se caso o dispositivo não encontrar a fonte Arial, ele buscará pela fonte Helvetica, ou sans-serif.
Podemos pesquisar no google por: "css web safe font combinations".

    • ID = (#) - identificação.
    • CLASS = (.) - classe

    REGRA: Em um mesmo documento HTML, só podemos usar um id para um único elemento, diferente do class.

    Pseudo-classes ex020

    Pseudo-classes são relacionadas ao estado de um determinado elemento. Elas são respresentadas pelos (:), asscom como a id é o (#) e as classes são o (.);

    • :hover - significa que quando passar o mouse por um elemento, acontecerá algo.
    • :visited
    • :active
    • :checked
    • :empty
    • :focus

    Exemplo de :hover - Passe o mouse por cima desse texto.

Informações sobre o que foi visto na aula

  • # = id
  • . = class
  • : = pseudo-class
  • :: = pseudo-element
  • > = children (filho)
